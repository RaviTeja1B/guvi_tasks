Object Reprasentation in javaScript

Objects

Object in javaScript can be solved using the Object Oriented Programming (OOP) Mechanisam.
An object is a collection of properties.
A property is an association b/w a key's and value's
We can add properties into curly brace {} as key: vlaue

Polymorphism : Polymorphism is one of the core concepts of object-oriented programming languages. Polymorphism means the same function with different signatures is called many times. In real life, for example, a boy at the same time may be a student, a class monitor, etc. So a boy can perform different operations at the same time. Polymorphism can be achieved by method overriding and method overloading

Accessing Object Properties

With the help of the DOT(.) Notation and BracketNotation[] we can access the object values.
For the BracketNotation we can get the output for the Invalid Key
For the DOT Notation we cannot get the output it will throw an error as Invalid Syntax.
Methods involve in the OOP construction are many among them Class method, Constructor Method, Object Method, Function Method are most used.
